Clase Tversion

clase_Version.pas
Clase Propiedad/Metodo Descripción Formato
Tversion class function NumeroVersion: string Funcion que devuelve la version del SDK en formato numerico. 00.00.000 => XX : Version Principal, YY: Subversion. ZZZ: Correcciones
Tversion class function TipoVersion: string; Funcion que devuelve el tipo de version El tipo de version es: Alpha, Beta, RC (Release Candidate), Final
Tversion class function FechaVersion: string Funcion que devuelve la fecha de la version Fecha en formato dd-mm-yyyy
Ejemplo de Uso

A continuación, un ejemplo básico de cómo usar la clase Tversion en Delphi:

        
            uses clase_version;
            Procedure version;
            begin
                Showmessage(Tversion.NumeroVersion);
            end;
